

### 1. ¿Qué es "Lo que sale de BERT"? (LOS DATOS / Variable)
**Analogía:** Es la **HOJA DE EXAMEN** individual de cada estudiante.
**Nombre Técnico:** **Embeddings Contextualizados (Hidden States)**.

*   Esto es lo que TinyBERT acaba de "cocinar".
*   Es el vector numérico de 768 números que representa a la palabra "tecnología".
*   **Es ÚNICO para cada estudiante.** El Estudiante 1 tiene el vector de "get_news", el Estudiante 2 tiene el de "tecnología".
*   **IMPORTANTE:** Esto **NO** es la base de datos (pesos). Esto es la **materia prima** que vamos a clasificar.

### 2. ¿Qué es "La Base de Datos Entera"? (EL MODELO / Fijo)
**Analogía:** Es la **PIZARRA GIGANTE** en la pared con las instrucciones.
**Nombre Técnico:** **Matriz de Pesos de la Capa de Salida ($W$)**.

*   Esta es la "inteligencia" que se quedó guardada después de entrenar.
*   Es una matriz de `[768 filas x 17 columnas]` (tus 17 etiquetas).
*   **Es COMPARTIDA.** Es idéntica para todos.
*   Esta matriz contiene "la lógica" que dice: *"Si el número en la posición 5 del vector es alto, entonces es probable que sea una FECHA"*.

### EL FLUJO EXACTO (Definición Clara)

Imagina el momento exacto del cálculo en la GPU para la palabra "tecnología".

**A. EL ESTUDIANTE (CUDA Core #5)**
El núcleo número 5 se despierta y dice: *"A mí me toca procesar la palabra 'tecnología'"*.
1.  **Recoge su Dato Personal:** Va a la memoria y agarra **EL VECTOR QUE SALIÓ DE BERT** correspondiente a "tecnología".
    *   *Dato:* Un vector de `[1 x 768]`.
    *   *Nota:* Este vector es diferente al del estudiante de al lado.

**B. LA CONSULTA A LA "BASE DE DATOS" (Lectura de Pesos)**
2.  **Mira la Pizarra (Broadcasting):** El núcleo levanta la vista y lee la **MATRIZ DE PESOS (W)** que es la misma para todos.
    *   *Dato:* La matriz completa de `[768 x 17]`.
    *   *Acción:* No se "descarga" una copia privada, simplemente accede a esa memoria compartida de alta velocidad.

**C. EL PROCESO (El Cruce)**
3.  **La Multiplicación:** El estudiante cruza SU vector personal (Output de BERT) contra LA matriz compartida (Pesos de Salida).
    *   `[Mi Vector de Tecnología]` **X** `[Matriz de Pesos Compartida]` = `[Mis Logits]`

| COMPONENTE | NOMBRE TÉCNICO | EN LA ANALOGÍA | ¿ES IGUAL PARA TODOS? |

| **INPUT** | **Contextualized Embedding** (Salida de BERT) | La Hoja de Examen del estudiante 

| **LÓGICA** | **Weights Matrix** (Capa Lineal) | **La Pizarra / Base de Datos** | **SÍ**. Es la referencia fija y "Solo Lectura". |

**Corrección Final a tu frase:**

No es que *"recoja de la base de datos la matriz entera"*.
Es que **USA** la matriz de pesos (la base de datos de conocimiento) para **TRANSFORMAR** su vector personal (lo que salió de BERT).
